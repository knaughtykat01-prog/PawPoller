# Platform Tag & Metadata Limits

## Tag Limits
| Platform | Max Tags | Format |
|---|---|---|
| Inkbunny | Unlimited | Comma-separated, underscores for spaces |
| SoFurry | 97 max | Comma-separated, spaces in tags OK |
| FurAffinity | Unlimited | Space-separated, underscores for multi-word |
| Wattpad | 24 max | Space-separated, camelCase, no underscores |
| SquidgeWorld/AO3 | Unlimited | Comma-separated freeform tags |
| Bluesky | N/A | No tags (announcements only) |
| DeviantArt | 30 max | Array |
| Itaku | 59 chars per tag, min 5 tags | JSON objects |

## SQW/AO3 Specific Fields

### Ratings
- Not Rated
- General Audiences
- Teen And Up Audiences
- Mature
- Explicit

### Archive Warnings
- Choose Not To Use Archive Warnings
- No Archive Warnings Apply
- Graphic Depictions Of Violence
- Major Character Death
- Rape/Non-Con
- Incest and/or Incestuous Relationship(s)
- Suicide/Suicidal Ideation
- Underage

### Categories (Relationship Types)
- F/F — Female/Female
- F/M — Female/Male
- Gen — No romantic/sexual focus
- M/M — Male/Male
- Multi — Multiple relationship types or multi-partner
- NB/F — Non-Binary/Female
- NB/M — Non-Binary/Male
- NB/NB — Non-Binary/Non-Binary
- QPR — Queer-platonic relationships
- Vs./Antagonistic — Adversarial relationships
- Other — Other relationships

### Relationship Notation
- `/` = Romantic (e.g. "Character A/Character B")
- `&` = Platonic (e.g. "Character A & Character B")
- `~` = QueerPlatonic
- `vs` = Antagonistic/Adversarial

### Fandom Tags
- Use full fandom names (e.g. "Kung Fu Panda", "Elder Scrolls")
- "Original Work" for non-fanfic
- Multiple fandoms separated by commas for crossovers

### Additional Tags (Freeform)
- Any descriptive tags (Angst, Crossover, Tentacles, etc.)
- Can include content warnings not covered by Archive Warnings
- Do NOT put fandom, relationship, or character names here
- Comma-separated
