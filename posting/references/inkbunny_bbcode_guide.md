# Inkbunny BBCode Reference

## Text Formatting
- `[b]bold[/b]` — **bold**
- `[i]italic[/i]` — *italic*
- `[u]underline[/u]` — underlined
- `[s]strikethrough[/s]` — ~~strikethrough~~
- `[t]title text[/t]` — title/heading style

## Alignment
- `[left]text[/left]` — left aligned
- `[center]text[/center]` — center aligned
- `[right]text[/right]` — right aligned

## Quotes
- `[q]text[/q]` — blockquote
- `[q=Username]text[/q]` — attributed quote

## Links
- `[url]www.example.com[/url]` — auto-linked URL
- `[url=http://example.com]Link Text[/url]` — named link

## Colors
- `[color=red]text[/color]` — named color
- `[color=#bada55]text[/color]` — hex color

## Unformatted
- `[code]text with [b]tags[/b][/code]` — shows raw BBCode without rendering

## Submission Thumbnails
Replace `submission_id` with the ID, optionally add `,page` for multi-page.
- `[smallthumb]3842922[/smallthumb]` or `#S3842922`
- `[mediumthumb]3842922[/mediumthumb]` or `#M3842922`
- `[largethumb]3842922[/largethumb]` or `#L3842922`
- `[hugethumb]3842922[/hugethumb]` or `#H3842922`

## Pools
- `[smallpool]pool_id[/smallpool]`
- `[mediumpool]pool_id[/mediumpool]`

## User Icons
- `[icon]KnaughtyKat[/icon]` — avatar icon
- `[iconname]KnaughtyKat[/iconname]` or `@KnaughtyKat` — icon + name
- `[name]KnaughtyKat[/name]` or `ib!KnaughtyKat` — name link only

## External Account Links
- `[da]username[/da]` or `da!username` — deviantART
- `[fa]username[/fa]` or `fa!username` — Fur Affinity
- `[sf]username[/sf]` or `sf!username` — SoFurry
- `[w]username[/w]` or `w!username` — Weasyl

## Notes for Story Formatting
- IB has NO `[hr]` tag — use `[center]───────────────────[/center]` for dividers
- IB has NO `[h1]`/`[h2]` tags — use `[t]` for titles or `[center][b]Title[/b][/center]`
- Body text should NOT use `[center]` — only headings, dividers, and `* * *` breaks
- Fully-italic paragraphs are normal body text, not subtitles — do not center them
